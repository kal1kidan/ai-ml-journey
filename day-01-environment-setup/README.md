# Day 1 — Environment Setup ✅

## Goal

Set up a professional Python development environment for my AI and Machine Learning journey.

## What I Completed

### Python Setup

- Installed Python 3.12
- Verified Python installation:

```bash
python --version
```

- Verified pip installation:

```bash
pip --version
```

---

## Virtual Environment

Created my first Python virtual environment:

```bash
python -m venv ai-env
```

Activated the environment:

```powershell
.\ai-env\Scripts\Activate.ps1
```

### Purpose

- Keep project dependencies isolated
- Maintain a clean development workflow
- Avoid conflicts between different projects

---

## Installed Core AI/ML Packages

Installed:

```bash
pip install numpy pandas requests python-dotenv
```

Packages:

- **NumPy** → Numerical computing and mathematical operations
- **Pandas** → Data analysis and manipulation
- **Requests** → Working with APIs and external data
- **Python-dotenv** → Managing environment variables securely
